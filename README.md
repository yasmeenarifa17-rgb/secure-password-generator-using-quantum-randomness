# secure-password-generator-using-quantum-randomness
This project Leverages Qiskit to generate high-entropy passwords using true quantum randomness. By harnessing qubit superposition and measurement collapse, this tool produces unpredictable, non-deterministic bitstrings mapped to secure, robust passwords. A modern cybersecurity approach outperforming classical pseudo-random generators. ⚛️💻

🔏 Quantum Secure Password Generator

This project was developed during a Quantum Computing Workshop conducted at our college.

💡 Project Idea
Traditional password generators rely on pseudo-random algorithms. This project uses quantum randomness to generate highly secure passwords that are unpredictable and resistant to attacks.
